# Background Music

Place your background music file here as `background-music.mp3`.

## Recommendations

- **Format**: MP3 (best browser compatibility)
- **Duration**: 2-4 minutes (will loop automatically)
- **Style**: Soft instrumental, romantic ambient, or acoustic
- **Volume**: Pre-normalized to avoid loud playback (the player sets volume to 30%)
- **Bitrate**: 128-192 kbps for good quality without large file size

## Suggested Music Styles

- Soft piano instrumentals
- Acoustic guitar melodies
- Classical string quartets
- Ambient romantic themes
- Traditional Arabic instrumental (Oud, Qanun)

## License
